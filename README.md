# react-materialui-boilerplate

## ディレクトリ構成
  * docs
  * src
  * tasks
  * test

**docs**  
[jsdoc](https://github.com/jsdoc3/jsdoc) で生成されたファイルを置きます。  

**src**  
ソースコードや画像、フォントなどのファイルを置きます。  

**tasks**  
タスクランナー用ファイルを置きます。  
[gulp](http://gulpjs.com/)  `npm i -g gulp-cli`

**test**  
テスト用ファイルを置きます。  
[Mocha](https://github.com/mochajs/mocha)  `npm i -g mocha`
